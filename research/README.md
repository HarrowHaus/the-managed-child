# research/ — the permanent evidence base

This directory is the project's **source of record for evidence**: research harvests
and operator rulings live here, versioned, next to the nodes they justify. The
writing pipeline (researcher → grader → carder → essayist → critic), the
`maintainer`, and every future session read from **here**, not from chat uploads
and never from memory.

## The law of this directory
- **Every node built from a harvest must cite its primary.** No `documented`
  standing without an attached confirming primary (the moat). This is enforced
  mechanically by the grade-vocab guard and the edge-lint; it is enforced
  editorially here.
- **Addenda win over parent rulings.** Where a `*-ADDENDUM.md` conflicts with its
  parent rulings file, the addendum governs.
- **Search before grading** (per `crowley-rail-RULINGS-ADDENDUM.md`): a standing is
  assigned only after its documented substrate has been searched for and found —
  never from priors, never "apocryphal because it pattern-matches conspiracy."
