# THE METHOD (reusable, separable from this application)
*The managed child is the first application of a method — not the method itself. Kept separate so it can be reused, taught, or applied to another contested cultural-history claim.*

## The method in five moves
1. **Grade every edge** (WELD / HYPOTHESIS / FICTION-ALERT). The grading is the argument.
2. **Reverberation, not conspiracy.** Influence = working-off, no coordination required; distinguish worked-off (encounter) from same-field (parallel).
3. **Symptom, not cause.** Include ugly beliefs as record; never route causation through a coded collective.
4. **Name the denominator.** State the base rate; concede what the thesis does NOT claim, first.
5. **Structure ≠ intent.** Separate documented craft from asserted meaning; individually grade every reading.

## Why it's a moat
The genre this lives in (esoteric / conspiracy analysis) is epistemically garbage. Being the rigorous one is a permanent differentiator. The discipline that feels like a constraint is the unique selling proposition — and it is portable to any topic where "hidden influence" is claimed.
