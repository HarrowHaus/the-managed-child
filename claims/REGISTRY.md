# claims/REGISTRY.md — the claim register (MASTER-SPEC 2.2)
*The project's load-bearing propositions, decomposed to the claim. Page grades derive from these rows; **credence lives here, not on the page** (AUDIT-3 II). This is also the conspiracy-compendium intake: every claim is split into documented events / attributed intent / apocrypha-with-a-career / suppressed-zone / open remainder, and only the honest remainder stays open. Per-node rows are populated in Phase 4 as each node is absorbed; this file holds the schema and the seed tranche.*

## Schema (one row per claim)
- **id** — `CLM-<node>-<n>`
- **proposition** — the exact claim, one sentence, falsifiable.
- **mechanism** — the edge-mechanism class it rests on (EDGE-VOCABULARY v2): direct-transmission · institutional-relay · network-exposure · common-inheritance · convergent-selection · analogy · (or *event* for a bare documented fact).
- **evidence for** — source + **location** (page/section/URL). No bare "PRIMARY".
- **evidence against** — what cuts the other way, if anything.
- **alternatives** — the two strongest non-thesis explanations.
- **search universe** — the widening rings actually searched (L3): published → correspondence/marginalia/libraries → employers/societies/intermediaries → grants/ledgers/minutes → litigation/oral-histories/counterparty-archives → variants/translations. Required for any negative.
- **standing** — confirmed · strongly-supported · unresolved · weakened · contradicted · apocryphal. (Derives the page grade; **weight/credence never moves it** — only a found primary flips it, R-12.)
- **reopen** — the condition that would change the standing.

## Standing ↔ grade map
confirmed/strongly-supported → **WELD** (documented) · unresolved/weakened → **HYPOTHESIS** (possible) · contradicted/apocryphal → **FICTION-ALERT** (apocryphal). A `suppressed-zone` tag (documented destruction in the claim's evidence radius) stops absence from counting against it and makes the National Security Archive's live routes (grant ledgers, university/hospital archives, contractor records, lawsuits, other-agency copies, oral histories) the mandatory next step — never a terminal "the files burned."
