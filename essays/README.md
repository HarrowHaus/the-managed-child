# Essays

**Scope (settled — see `../DECISIONS.md`): essays are essays *about the work itself*** —
the thesis, the lineage, and the method. Each draws on `data/nodes/**` (and where
relevant `THESIS.md`, `map/lineage-map.md`), links its claims back to the reference
nodes that ground them, and is built and cross-linked by the site.

These are **not** media decodes. The "Ask About Illuminati" children's-media decode
series (Finding Nemo / Dory, Cars, Fantasia, Inside Out, Pokémon, Harry Potter,
Nickelodeon) is **parked** in [`_parked-decodes/`](_parked-decodes/) — out of the
active build for now, to be reintroduced later.

## What belongs here

- Essays that argue, explain, or interrogate the project's own material: the root
  impulse, a rail, the four-idiom through-line, the lifespan continuum, the
  grading method, a specific documented lineage.
- Sober or inhabited voice as the piece calls for — but the subject is the work,
  not a film.
